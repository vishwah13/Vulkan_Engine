# Vulkan_Engine
A graphics engine using Vulkan API and C++

## Build dependencies
* CMake 3.8
* Vulkan SDk

## Current progress

![engine_ikiuLbHRi8](https://github.com/user-attachments/assets/eeae8afa-1dfc-47f8-8a47-c5963756ab41)

## Features

* Dynamic rendering
* glTF scene loading
* Frustum culling
* Scene Graph
* Compute Shader
* Bindless textures and samplers
